"Hits the Spot" Spotify Playlist Generator

  1. Description: Spotify Playlist Generator Engine that runs on various input forms such as a desired moods, intended occasions, sample songs, etc. to create a curated playlist.
  2. Usefulness: Not everyone is blessed with impeccable music taste. For multiple reasons, be they time, ability, niche, people can't always make their own playlists, but there will always be a need for a great playlist to supply the atmosphere for any occasion. "Hits the spot" allows for even the most uninformed playlist desirer to concoct a great playlist. Similar smaller project ideas exist, however they are rather limited in the number of parameters they can recieve, and are largely dependent on the user's existing listneing habits. Our application, aside from using the spotify API, will utilize web scraping of different online contexts to categorize songs more profoundly and be able to generate playlists from a wider set of parameters.
  3. Realness: The majority of our data will come from the spotify api and will consist mostly of song metatata such as artists, views, genres, etc, but a significant portion of our data will also come from web scraping that will gather where songs have been found and attributed to
  4. Functionality:
    • Basic Functions:
      • Insert: Upon a new release, for example, our app regularly will fetch from the spotify API and categorize it based on its metadata to later be searched/queried
      • Update: Not regularly used as songs are not changed post release, but should a song need to be removed (for example taken off platform), a song can be updated and/or removed
      • Delete: Same as above
      • Search: Majority of the function of the database: given various query parameters, our application will return a list of songs with the most similitude in order to fit the desired vibe/requirement/function
    • Advanced Functions:
      • Song Fit: Given an existing playlist that a user enters, our database technology will determine how similar a song is to the remainder of the playlist and whether it is a good fit
      • Playlist Manicurer: Given an existing playlist, our app will determine which playlist songs are the most dissimilar/similar to each other and give suggestions for which songs to drop in order preserve the "vibe" of the playlist
